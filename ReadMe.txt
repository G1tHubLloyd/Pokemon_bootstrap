# Pokémon Modal Viewer 🐾

A simple web app that displays a list of Pokémon and shows detailed information in a Bootstrap-powered modal. Built using HTML, CSS, JavaScript, and the PokéAPI.

---

## 🚀 Features

- Fetches and displays the first 150 Pokémon from the PokéAPI
- Clickable list items open a modal with:
  - Pokémon name
  - Sprite image
  - Height
- Responsive design using Bootstrap 5
- Clean and accessible layout
